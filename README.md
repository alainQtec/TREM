# [TREM](https://www.powershellgallery.com/packages/TREM)

Throatware Removal Module

![Image](https://github.com/user-attachments/assets/9b5e5912-9379-4fc5-9991-c89e8703e4d5)

## Usage

```PowerShell
Install-Module TREM
```

then

```PowerShell
Import-Module TREM
# do stuff here.
Remove-Bloat -c ads, network -Force
```
